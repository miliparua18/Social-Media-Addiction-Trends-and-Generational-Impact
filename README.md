# 📊 Social Media Addiction Trends and Generational Impact

A data-driven web application that analyzes and predicts **social media addiction levels** and forecasts **future trends** using Machine Learning and Time Series Forecasting.

---

## 🚀 Overview

This project focuses on understanding the impact of social media usage by combining:

- 🧠 User-level addiction prediction  
- 📍 City & state-based analytics  
- 🔮 Future trend forecasting using Exponential Smoothing  
- 📊 Interactive data visualization  

---

## 🎯 Key Features

### 🔹 Addiction Level Prediction
- Users answer a questionnaire based on their social media habits
- The system predicts:
  - Low Addiction
  - Medium Addiction
  - High Addiction

---

### 🔹 Area-Based Visual Insights
- Select **State and City**
- Visualizations include:
  - Addiction distribution
  - Age-based analysis
  - Education-level analysis
  - User vs city comparison (screen time & anxiety)

---

### 🔹 Future Trend Forecasting 🔮
- Predicts addiction trends for upcoming years
- Uses **Exponential Smoothing**
- Provides:
  - Year-wise predictions
  - Trend visualization
  - Growth patterns

---

### 🔹 Personalized Insights
- Helps users understand their digital behavior
- Provides awareness about addiction risks

---

## 🛠️ Tech Stack

### 💻 Frontend
- React.js  
- Chart.js  
- CSS  

### 🔙 Backend
- Flask (Python)  
- REST API  

### 🤖 Machine Learning & Forecasting
- Random Forest
- Scikit-learn  
- Pandas, NumPy  
- Statsmodels (Exponential Smoothing)  

---

## ⚙️ How It Works

1. User inputs behavioral data through a questionnaire  
2. ML model predicts addiction level  
3. City/state data is used for comparative analysis  
4. Historical data is processed using Exponential Smoothing  
5. Future predictions are generated and visualized

## ▶️ Running the Project

cd Social-Media-Addiction-Trends-and-Generational-Impact

### 2. Run Backend

### 3. Run Frontend
cd frontend<br>
npm install<br>
npm start<br>
